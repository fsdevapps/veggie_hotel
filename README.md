VEGGIE HOTEL
-----------------
//Concept
 App for Veggie Hotel Project that allows to manage all about it

//Features
 -> Login: verify employee authenticity
 -> Bookings: register new bookings and manage them
 -> Guests Entry/Exit: manage when guests enter or leave the hotel
 -> Schedule: view hotel activities in a schedule
 -> Contacts: get in touch with hotel
 -> Location: view hotel location
 -> Stats: some stats about the hotel

//Data Storage
 All the data is storaged on a Database

//Used Languages
 The languages used on this app were C# and SQL

//Target Users
 This app is built for Veggie Hotel employees/managers
